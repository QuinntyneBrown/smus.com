Building cross-device webapps
=============================
type: link
link: http://www.html5rocks.com/mobile/cross-device
posted: 2012-04-24

I wrote an [HTML5Rocks][h5r] article about [building cross-device
webapps][article]. The gist of the approach is to use client-side
feature detection using [device.js][], and then switch to server side UA
detection if performance becomes an issue.

[article]: http://www.html5rocks.com/mobile/cross-device
[device.js]: https://github.com/borismus/device.js
